# 웹 페이지 → 한국어 마크다운 변환

1. URL의 웹 페이지 내용을 가져온다.
   - `web_fetch` 사용을 우선하며, 실패 시 `google_web_search`로 대체 검색하거나 `curl -sL "https://web.archive.org/web/2025/URL"`을 사용한다.
2. 본문 전체를 생략이나 요약 없이 한국어로 상세히 번역한다. (예시, 코드, 대화문, 보너스 섹션 등 모든 내용을 포함)
3. 번역문 최상단 제목 바로 아래 줄에 `원문: [URL](URL)` 형식으로 링크를 추가한다.
4. `/tmp/` 에 `영문-제목.md` 파일로 즉시 저장한다. (원문의 영문 제목을 소문자로, 단어는 -로 연결)
